## Problem Description

---

## Examples

### Example 1

```
Input:
matrix = [
  [3, 5],
  [1, 9]
]

Output:
Max = 9, Min = 1

Explanation:
9 is the largest and 1 is the smallest element.
```

### Example 2

```
Input:

Output:
Max = -5, Min = -5

Explanation:
Single element matrix.
```
