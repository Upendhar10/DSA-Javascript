## Problem Title

### Transpose of a Matrix

---

## Problem Description

Given a matrix, return its **transpose**, where rows become columns.

### ⚠️ Edge Cases to Consider

* Empty matrix
* Single row or column
* Square and rectangular matrices

---

## Input Format

* A matrix `matrix`

---

## Output Format

* Transposed matrix

---

## Examples

### Example 1

```
Input:
matrix = [[1, 2, 3]]

Output:
[[1], [2], [3]]

Explanation:
columns become rows.
```

---

## 📏 Constraints

* `0 ≤ rows, cols ≤ 100`

---

## 📝 Notes ( if any )

* New matrix dimensions swap
