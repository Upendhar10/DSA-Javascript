## Problem Title

### Find Upper and Lower Triangle Sum

---

## Problem Description

Given a **square matrix**, calculate the **sum of upper triangle elements** and **lower triangle elements**.

### ⚠️ Edge Cases to Consider

* Empty matrix
* 1×1 matrix
* Non-square matrix

---

## Input Format

* Square matrix `matrix`

---

## Output Format

* Upper triangle sum and lower triangle sum

---

## Examples

### Example 1

```
Input:
matrix = [
  [1, 2],
  [3, 4]
]

Output:
Upper Sum = 7
Lower Sum = 8

Explanation:
Upper: 1 + 2 + 4
Lower: 1 + 3 + 4
```

---

## 📏 Constraints

* Matrix must be square

---

## 📝 Notes ( if any )

* Diagonal elements counted in both
